- 主字体：`'PingFang SC', 'Microsoft YaHei', sans-serif`

## 4. 间距规范

- 容器最大宽度：600px
- 内边距：20px / 30px / 40px
- 区块间距：25px

## 5. 响应式断点

- 移动端优先设计
- 容器自适应，最大宽度 600px

## 8. 内容文案规范（东北雨姐风格）

### 词汇对照表

| 规范写法 | 东北话写法 |
|----------|------------|
| 汗脚 | 汗脚 |
| 保持清洁 | 整干净 |
| 透气 | 嘎嘎透气 |
| 非常好 | 嘎嘎好使 |
| 坚持 | 整起来 |
| 必须 | 必须 |
| 每天 | 天天 |
| 可以 | 整点 |
| 避免 | 别整那 |

### 常用语气词

- 整 - 搞、做
- 嘎嘎 - 非常、特别
- 贼 - 很、非常
- 老 - 很、特别（老不得劲）
- 那玩意儿 - 那东西
- 整一下子 - 弄一下
- 干就完了 - 做就对了

### 引以为戒的写法

❌ 不要：`每天用温水泡脚，保持清洁`
✅ 要：`天天温水泡泡脚，使劲整干净`
