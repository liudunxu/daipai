## SEO 流量提升计划

### 问题诊断 (2026-Q2)

#### 严重问题（立即修复）

| 问题 | 影响 | 优先级 |
|------|------|--------|
| sitemap 缺失 20+ 页面 | 搜索引擎未收录 | P0 |
| 页面 metadata 不完整 | 长尾关键词丢失 | P0 |
| 'use client' 页面无法 export metadata | SEO 效果打折 | P1 |

#### 基础问题（持续优化）

| 问题 | 说明 | 状态 |
|------|------|------|
| robots.txt | 已配置 Google/Baidu/Sogou/360 | ✅ |
| sitemap.xml | 需补充完整 | 🔄 |
| 结构化数据 | 已有基础 Schema | 🔄 |
| 页面速度 | 需监控 Core Web Vitals | 🔄 |

### 修复任务

- [ ] **sitemap 补全**: 补充缺失页面（github-rank, claude-code-leak, harness, todayinhistory, 等）
- [ ] **页面 metadata 审计**: 逐个检查并补充所有页面的 SEO metadata
- [ ] **'use client' 页面处理**: 将 metadata 分离到 server component 或使用 Head 组件
