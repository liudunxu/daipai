# AGENT.md - Claude Code 行为规范

本文档定义 Claude Code 在本项目中的行为规范。

## 项目信息

- **项目名称**: 极客观察
- **类型**: Next.js 14 多功能工具类网站
- **详细规范**: 见 [CLAUDE.md](CLAUDE.md)

## 核心规则

### 页面管理
- **新增或删除页面时，必须同步更新 `/nav` 导航页面的链接**
- 保持导航页面包含所有有效页面的入口

### Git 提交
- 每次修改完成后自动提交
- 提交格式: `feat: xxx`, `fix: xxx`, `style: xxx`, `refactor: xxx`, `docs: xxx`
- 提交信息使用中文描述

### 构建验证
- **推送前必须本地构建通过**: `npm run build`
- 如有错误自动修复后再次构建验证

## 任务判定

| 条件 | 处理方式 |
|------|----------|
| 涉及 3+ 文件修改 | 进入 plan 模式 |
| 新增页面/路由/API | 进入 plan 模式 |
| 引入新依赖 | 进入 plan 模式 |
| 样式修改、Bug 修复、文案更新 | 直接执行 |

## 注意事项

1. 外部图片建议使用本地图片或 base64
2. JSON 文件数据是临时的，生产环境使用 Vercel KV
3. 儿童类工具优先使用选择器，减少手动输入
