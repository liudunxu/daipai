# 汗脚烦恼 - 东北雨姐推荐

一个关于足部护理的营销推广单页，风格采用东北雨姐的口气。

## 快速开始

直接在浏览器中打开 `hanjiao.html` 即可预览。

```bash
# Mac
open hanjiao.html

# Windows
start hanjiao.html
```

## 页面预览

- 顶部：东北雨姐头像 + 推荐标签
- 内容：汗脚困扰 + 解决方法 + 日常小贴士
- 结尾：行动召唤

## 技术栈

- HTML5
- CSS3（内联样式）
- 无需安装依赖
